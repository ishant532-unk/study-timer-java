# Smart Study Timer (Anti-Distraction System)

## 📌 Description

Smart Study Timer is a Java-based command-line tool designed to help users track their study sessions and improve productivity. It records the time spent studying and stores logs for analysis, helping build discipline and consistency.

---

## 🎯 Problem Statement

Students often face:

* Poor time management
* Lack of proper study tracking
* Frequent distractions

This leads to reduced productivity and inefficient study habits.

---

## 💡 Solution

This project provides a simple CLI-based system that:

* Tracks study session duration
* Stores study data in a file
* Helps analyze daily study performance

---

## ⚙️ Features

* Start and stop study sessions
* Automatic calculation of study time
* File-based data logging
* Lightweight and easy-to-use interface

---

## 🛠️ Technologies Used

* Java
* File Handling
* Java Time API (LocalTime, Duration)

---

## 🚀 How to Run

1. Clone the repository
2. Open in any Java IDE (VS Code / IntelliJ / Eclipse)
3. Compile and run `StudyTimer.java`
4. Use menu options to control study sessions

---

## 📂 Output

* Study logs are stored in `study_log.txt`
* Each entry records total time studied
